VNL.dev Root Structure
=====================

/vnl.dev/
├── public/                      # Public facing assets
│   ├── index.html              # Homepage
│   ├── about.html              # About page
│   ├── contact.html            # Contact page
│   ├── search.html             # Search interface
│   ├── docs.html               # Documentation
│   ├── css/                    # Stylesheets
│   ├── js/                     # JavaScript files
│   └── assets/                 # Images, fonts, etc.

Security:
- SSL/TLS enabled
- API rate limiting
- Input validation
- XSS protection
- CSRF protection

Access Control:
- Owner: Full access
- Admin: System management
- Moderator: Content management
- Helper: Support access
- User: Basic access

Version Control:
- Git repository
- Protected branches
- Automated CI/CD
